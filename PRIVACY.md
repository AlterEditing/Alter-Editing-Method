# Privacy Notice

## What the app sends

The app may send requests to:

- Telegram-related auth flow endpoints
- GitHub API / GitHub Releases for update checks and downloads

## What may be processed

- authorization session identifiers
- Telegram ID returned by auth backend
- app version and update metadata

## What is stored locally

Local settings can include:

- language/theme preferences
- auth state fields used for session continuity

## Data retention

Retention and server-side handling are defined by the backend service operators.
This repository contains the desktop client only.
